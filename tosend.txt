How do I...

>...create new node?
>>Click on the node which you want to make parent node, hold left mouse button down and release button under the green icon. Using mouse, 'Shift' and 'Ctrl' (*) keys put created node into preference position and click left mouse button again. Then, you can type name of the new node (You can left name field if you want)(*).

>...edit existing node?
>>Click on the node which you want to edit, hold left mouse button down and release button under the blue icon, or just click on the node with right mouse button(*). Select action (move node arround his parent, edit node's data or link branch to other node. If you will select 'edit', in appeared window you can set name and text of the editing node. 

>...delete existing node?
>>Be careful: Deleting node with children causes deleting of branch
>>Click on the node which you want to delete, hold left mouse button down and release button under the red icon. Confirm appeared question.


* - if you had not been editing the configuration file

txt11://Аннотация
Trees - приложение для ведения и организации заметок. Позволяет создавать древовидные структуры из узлов, способных хранить текстовую информацию. 
Возможности:
> Редактор графов, позволяющий распологать узлы как на строго заданном растоянии, под фиксированными углами, так и свободно
> Сохранение/загрузка деревьев в файл
> Возможность перепривязки веток к другому узлу
